Lovelang

Lovelang is a chrome extension that replaces words on a webpage into a new language so that the user can learn the language while surfing the web! The word to replace and the number of replaced words depend on the frequency of the word on the page and the user's learning speed respectively.

Built using Google Translate API, Javascript, Express and MongoDB.
